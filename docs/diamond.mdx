We will run the bandstructure calculation from the OpenMX example file. Here is
the input file:

import diamond_input from '!!raw-loader!/src/diamond/diamond_input.txt';

<CodeBlock className="fortran" title="src/diamond/diamond_input.txt">{diamond_input}</CodeBlock>

We can run the calculation using mpirun or submit via job-script:
```bash
mpirun -np 12 openmx diamond_input.txt > diamond_output.txt
```

Next step is to extract bandstructure data from `diamond.Band` file:
```bash
bandgnu13 diamond.Band
```

Finally, we can plot the bandstructure using gnuplot or any program of your
choice:
```bash
gnuplot diamond.GNUBAND
```
